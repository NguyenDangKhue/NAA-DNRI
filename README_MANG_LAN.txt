========================================
    CHIA SẺ LAB MANAGE QUA MẠNG LAN
========================================

🚀 CÁCH KHỞI ĐỘNG:
   • Windows: Double-click start_server.bat
   • Linux/Mac: ./start_server.sh
   • Thủ công: python -m app

🌐 TRUY CẬP:
   • Từ máy này: http://localhost:5000
   • Từ thiết bị khác: http://192.168.1.201:5000

📋 KIỂM TRA:
   • Xem thông tin mạng: python show_network_info.py
   • Test kết nối: python test_connection.py

⚠️  LƯU Ý:
   • Đảm bảo tất cả thiết bị cùng mạng WiFi
   • Kiểm tra Firewall nếu không truy cập được
   • Địa chỉ IP có thể thay đổi khi khởi động lại

📖 Chi tiết: Xem file HUONG_DAN_MANG_LAN.md
